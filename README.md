# Real Estate Price Prediction using Linear Regression

## Overview
This project predicts real estate prices using linear regression. It provides data analysis, model training, performance metrics, and comprehensive visualizations.

## How to Run

1. Ensure you have Python 3.8+ and pip installed.
2. Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. Place your dataset (e.g., `real_estate_prices.csv`) inside the `data/` directory.

4. Run the analysis script:

    ```bash
    python src/regression_analysis.py
    ```

## Project Structure

```
real-estate-price-prediction/
```

## Features

- Data preprocessing and encoding
- Linear regression modeling
- Training & test performance metrics (RMSE, MAE, R²)
- Feature importance and detailed analysis
- Rich plotting dashboard

## Visualization Dashboard

The script generates multiple plots showing model performance, errors, and feature analysis.
